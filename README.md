# Обеспечение качества КС

Репозиторий с практическими работами по дисциплине «Обеспечение качества функционирования компьютерных систем».

## Навигация

- [Практическая 4](<./Практическая 4/README.md>) — баг-репорты по сайту `nke.ru`
- [Практическая 5](<./Практическая 5/README.md>) — работа с TestIT
- [Практическая 6](<./Практическая 6/README.md>) — тестирование с помощью DevTools
- [Практическая 7](<./Практическая 7/README.md>) — работа с Postman

## Структура

Для каждой практической используется единый формат:

- в корне папки практической находится только `README.md`
- изображения лежат в `images/`
- данные и исходные файлы лежат в `data/`
- скрипты лежат в `scripts/`
- результаты и сгенерированные артефакты лежат в `results/` 
- дополнительные файлы лежат в `files/`
