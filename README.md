## Структура

Для каждой практической используется единый формат:

- в корне папки практической находится только `README.md`
- изображения лежат в `images/`
- данные и исходные файлы лежат в `data/`
- скрипты лежат в `scripts/`
- результаты и сгенерированные артефакты лежат в `results/` 
- дополнительные файлы лежат в `files/`
